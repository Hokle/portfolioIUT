Projets Web

iut_link

Ce projet est un réseau social développé pour connecter les étudiants de l'IUT. La plateforme permet de créer un compte, personnaliser son profil avec une photo, créer et s'abonner à des threads thématiques. Les utilisateurs peuvent partager des messages, des émojis et des images. Toutes les données sont stockées dans une base PostgreSQL. Le site intègre un système de messagerie en temps réel et de notifications. Ce projet a été réalisé en équipe de 2 personnes sur une durée de 3 semaines.

Compétences techniques acquises :

HTML/CSS JavaScript Node.js PostgreSQL APIs RESTful

Savoir-être acquis :

Travail en équipe Communication Gestion de projet Résolution de problèmes

Chasse au monstre

Ce projet est un jeu de chasse au monstre, C'est un jeu en multijoueur (même jouable en solo). Dans ce jeu un chasseur doit tirer sur un monstre qui lui tente de s'échapper. Tout en sachant que la partie est paramétrable. La carte de jeu possède un labyrinthe généré aléatoirement allongeant le chemin du monstre. Ce projet a été réalisé à 2 en 1 mois.

Compétences techniques acquises :

Java Architecture MVC Génération procédurale Interface graphique

Savoir-être acquis :

Collaboration Communication Planification
